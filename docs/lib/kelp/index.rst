Kelp API
========

Kelp currently includes a library of helpers for Capybara_. Future releases may
include support for Webrat_ or other popular Cucumber testing libraries.

.. _Capybara: http://github.com/jnicklas/capybara
.. _Webrat: https://github.com/brynary/webrat
